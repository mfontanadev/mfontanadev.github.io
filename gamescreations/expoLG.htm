L-Garden

Concepto:

    L-Garden es la combinación de Sistemas-L aplicado a estructuras arbóreas. Se llaman Sistemas-L, (la L es por Aristid Lindermayer su creador,  a quien debo pedirle disculpas por haber usado su creación para mis fines), es un lenguaje en base a símbolos y reglas. Una definición más detallada podemos encontrarla en http://es.wikipedia.org/wiki/Sistema-L.

Como pasar de los Sistemas-L a un "árbol":

    Para nuestro caso lso elementos del lenjuage que utilizaremos son: símbolos, semilla y reglas. La semilla es una secuencia de símbolos a la cual se le aplican reglas para obtener una nueva semilla. Luego, los elementos de la semilla pueden ser dibujados de manera que se obtenga una estructura en forma de árbol.

    A continuación se explican brevemente los pasos para pasar de los elementos del sistema L a una representación 3D:

  1. Definición de elementos: lo primero que debemos hacer es elegir los elementos que formarán nuestro àrbol 3D (ver figura de la izquierda).

  2. Genetic parser: este algoritmo se encarga de transformar los elementos en el código genético, el cual inicialmente es igual a la semilla, pero puede pasar que las reglas se apliquen una y otra vez para obtener un código genético más elaborado.

  3. Representación del código genético:  la idea es armar un graficador que reemplace cada símbolo por una imagen. Algunos símbolos pueden ser utilizados para  cumplir con acciones en vez de ser imágenes, por ejemplo los "[" indican que debemos rotar la imagen un ángulo determinado.

  4. 3D Engine:  graficador que se encarga de representar el código genético en forma tridimensional.

L-Garden aplicación:

    La aplicación adjunta a esta documentación permite ver la representación tridimensional de los códigos genéticos preestablecidos. Nota: para salir de la aplicación pulsar tecla ESC.

    Para comenzar a utilizar la aplicación hay que ejecutar el archivo L-Garden.exe, (puede ser necesario instalar DirectX 8 o 9). La pantalla inicial muestra cuatro recuadros (indicadores) con dos botones cada uno y un recuadro mayor con la representación 3D del código genético actual. La idea es utilizar los indicadores para variar parámetros del código genético.  

    A continuación se explica la funcionalidad de cada uno de los indicadores:

:modifica la longitud de las ramas.

:modifica la edad (veces que se aplican las reglas a la semilla) del árbol.

:selecciona un código genético diferente.

:modifica el angulo de abpertura de las ramas.
